写作介绍

Hexo是一个简单、快速、强大的基于 Github Pages 的博客发布工具，支持Markdown格式，有众多优秀插件和主题。
安装及初始化教程。

Sublime Text 3作为一款具有代码高亮、语法提示、自动完成且反应快速的编辑器软件，不仅具有华丽的界面，还支持插件扩展机制，对初学者十分友好，是Coding和Writing的不错选择。
